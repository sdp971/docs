## Obtenez les outils de développement dont vous aurez besoin

### Terminal

Vous utiliserez une **ligne de commande (terminal)** pour créer votre projet Astro et exécuter des commandes clés pour construire, développer et tester votre site.

Vous pouvez accéder à la ligne de commande via un programme terminal local adapté à votre système d'exploitation. Les applications courantes sont **Terminal** (MacOS/Linux), **Invite de commandes** (Windows) et **Termux** (Android). L'une de ces options sera probablement déjà sur votre machine.

### Node.js

Pour qu'Astro fonctionne sur votre système, vous devrez également avoir installé [**Node.js**](https://nodejs.org/fr/) en version `v16.12.0` ou ultérieure.

Pour vérifier si vous avez déjà une version compatible installée, exécutez la commande suivante dans votre terminal :

```sh
node -v

// Exemple de sortie
v16.14.0
```

Si la commande renvoie un numéro de version supérieur à `v16.12.0`, vous êtes prêt !

Si la commande renvoie un message d'erreur comme « Commande 'node' introuvable » ou un numéro de version inférieur à `v16.12.0`, vous devez [installer une version compatible de Node.js](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

### Éditeur de code

De plus, vous devrez télécharger et installer un **éditeur de code** pour écrire votre code.

:::tip[Nous utiliserons...]
Ce tutoriel utilisera **VS Code**, mais vous pouvez utiliser n'importe quel éditeur fonctionnant sur votre système d'exploitation.
:::

1. [Téléchargez et installez VS Code](https://code.visualstudio.com/#alt-downloads) ou un autre éditeur de code de votre choix.
